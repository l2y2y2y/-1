# 重活一遍 · 人生模拟器

一个纯前端的轻量人生模拟器 MVP。

## 功能

- 随机出生背景
- 抽取并选择天赋
- 分配初始属性
- 按年龄推进人生事件
- 关键事件选择与属性结算
- 结局报告与分享文案
- 本地收藏人生报告

## 本地打开

直接打开 `index.html` 即可运行。

也可以在当前目录启动本地服务：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

这是静态网页项目，不需要构建命令。
